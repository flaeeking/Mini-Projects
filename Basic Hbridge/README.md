# Basic H-Bridge Car

## Overview

This project is a simple Arduino-based car built to understand the working principle of an H-bridge motor driver. The car could move both forwards and backwards by reversing the polarity of the voltage supplied to the DC motors.

## Features

- Forward movement
- Reverse movement
- H-bridge motor control
- Basic DC motor interfacing

## Hardware Components

- Arduino Board
- H-Bridge Motor Driver
- DC Motors
- Chassis and Wheels
- Battery Pack
- Breadboard and Wiring

## Learning Outcomes

Through this project, I gained an understanding of how H-bridges reverse motor direction by changing the polarity applied to a motor and learnt the fundamentals of motor drivers and embedded motor control.
